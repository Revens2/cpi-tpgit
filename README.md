# TaskManager CPI

Application web de gestion de tâches projet — Formation Git Bachelor CPI

## Description

TaskManager CPI permet à une équipe projet de créer, organiser et suivre
ses tâches de développement. Les données sont persistées localement via
le localStorage du navigateur.
